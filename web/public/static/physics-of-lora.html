1. The CompuFlair Framework

CompuFlair's central thesis is that the entire field of ML can be interpreted through one equation from statistical mechanics:

$$P = \frac{e^{-F}}{Z}$$

where $F$ is the free energy (the loss or cost function), and $Z = \sum_{\text{configs}} e^{-F}$ is the partition function — a normalization over all possible configurations of the system. Training an ML model is equivalent to cooling a physical system: finding the low-energy configuration of parameters that best fits the data.

This document applies that lens at two scales — macroscopic and microscopic — and shows that they tell a single coherent story.

Part I — The Outside View

2. The Base Model as Thermodynamic Equilibrium

Llama 3.2 3B was pre-trained on trillions of tokens by Meta. In the CompuFlair framework, this pre-training was a massive annealing process: the system ($\sim\!3 \times 10^9$ parameters) was heated (high learning rate, large batches, diverse data) and slowly cooled over weeks of training on GPU clusters, settling into a low-energy equilibrium across the loss landscape.

The result is a parameter configuration $\mathbf{W}_0$ that sits in a deep, broad basin of the energy surface. The model's “knowledge” — grammar, reasoning patterns, factual recall — is encoded in the geometry of this basin.

Key insight: The base model is a solved system. Meta spent millions of dollars and weeks of GPU-cluster time carefully annealing 3 billion parameters into this equilibrium. It knows grammar, reasoning, conversation, world knowledge — an enormous amount of structure encoded in the curvature of that energy landscape.

Now you want it to also know about hardware diagnostics. The brute-force approach — full fine-tuning — would unfreeze all 3 billion parameters. In thermodynamic terms, you'd be melting the entire system down and re-annealing from scratch.

You don't melt down a cathedral because you want to add a room. Don't melt it — perturb it.
Figure 1: Full fine-tuning melts the entire energy landscape. LoRA keeps it frozen and applies a small, targeted perturbation.

3. LoRA as Perturbation Theory

In quantum mechanics and statistical mechanics, perturbation theory handles problems of the form:

$$\hat{H} = \hat{H}_0 + \lambda \hat{V}$$

where $\hat{H}_0$ is the solved Hamiltonian, $\hat{V}$ is a small perturbation, and $\lambda$ controls the perturbation strength. LoRA does exactly this:

$$\mathbf{W} = \mathbf{W}_0 + \Delta\mathbf{W} = \mathbf{W}_0 + \mathbf{B}\mathbf{A}$$
  • $\mathbf{W}_0$ (frozen base weights) = the ground state $\hat{H}_0$
  • $\Delta\mathbf{W} = \mathbf{BA}$ (low-rank adapters) = the perturbation correction $\lambda\hat{V}$
  • $\mathbf{B} \in \mathbb{R}^{d \times r}$, $\mathbf{A} \in \mathbb{R}^{r \times k}$, rank $r \ll \min(d, k)$
ParameterValuePhysics Analogy
Rank $r$88 perturbation modes per adapted layer
Base weights $\mathbf{W}_0$$\sim\!3 \times 10^9$, frozenGround state — solved, don't touch
LoRA parameters$\sim\!3.47 \times 10^6$ trainablePerturbation corrections only
Compression$\sim\!1500 : 1$Only 1 in 1500 degrees of freedom moves
Why this works: The “distance” in parameter space between the base model and our hardware-diagnostics specialist lies in a very low-dimensional subspace. Perturbation theory predicts that first-order corrections capture most of the effect when the perturbation is small relative to the unperturbed system. Higher-order corrections (higher rank, more layers) yield diminishing returns — just as in physics.
Figure 2: LoRA constrains the update to a low-rank subspace (green). The base model $\mathbf{W}_0$ stays fixed. Full fine-tuning (red) wanders through all 3B dimensions.

4. Training Dynamics as Annealing

The gradient descent process maps directly to thermodynamic annealing:

4.1 Learning Rate as Temperature

$$\eta \;\longleftrightarrow\; T$$

Our $\eta = 10^{-5}$ is very low — a cold annealing. The system almost never accepts uphill moves: $P(\text{accept}) \propto e^{-\Delta F / T}$. Appropriate for refining a perturbation to an already-good solution.

4.2 Batch Size as Thermal Noise

$$\text{batch size} \;\longleftrightarrow\; \frac{1}{\sigma^2_{\text{thermal}}}$$

Batch size 1 means high stochastic noise — each gradient update is computed from a single example. These thermal fluctuations help the perturbation parameters explore the low-dimensional LoRA subspace without destabilizing the solution.

Figure 3: The three thermodynamic phases of LoRA training. 600 iterations at batch 1 over 385 examples (∼1.95 epochs).

5. 4-Bit Quantization as Literal Quantization

This mapping is the most satisfying because it's not an analogy — it's the same operation:

$$\text{Continuous weight space } \mathbb{R} \;\longrightarrow\; 16 \text{ discrete levels} \quad (4\;\text{bits} = 2^4 = 16)$$

Each parameter is mapped to one of 16 discrete levels, like energy quantization in a quantum harmonic oscillator ($\Delta E = \hbar\omega$). It works because the base model is a low-temperature system — parameters cluster near equilibrium values, so 16 levels per parameter captures the meaningful structure with negligible loss. Memory drops from $\sim\!6$ GB to $\sim\!2$ GB.

Figure 4: Quantization maps a continuous weight distribution to 16 discrete levels. Minimal loss because the system is already at equilibrium.
Part II — The Inside View

Choice Question Our Config Physics Analogy
Rank $r = 8$ How many independent correction modes per layer? 8 modes Normal mode truncation
Layers top 8 of 32 Where in the network to apply the correction? Layers 25–32 Spatially localized perturbation

7. Within Each Layer: Normal Mode Analysis

7.1 The SVD Connection

Any matrix decomposes via the Singular Value Decomposition:

$$\mathbf{M} = \mathbf{U\Sigma V}^\top = \sigma_1 \mathbf{u}_1\mathbf{v}_1^\top + \sigma_2 \mathbf{u}_2\mathbf{v}_2^\top + \sigma_3 \mathbf{u}_3\mathbf{v}_3^\top + \cdots$$

where $\sigma_1 \geq \sigma_2 \geq \sigma_3 \geq \cdots$ are the singular values (the “energies” of each mode), and $\mathbf{u}_i, \mathbf{v}_i$ are the mode shapes. This is structurally identical to a normal mode expansion of a vibrating system: any complex motion decomposes into independent modes, each with its own energy and shape.

7.2 LoRA as Truncated Normal Mode Expansion

The correction $\Delta\mathbf{W} = \mathbf{BA}$ has rank at most $r = 8$:

$$\Delta\mathbf{W} = \mathbf{BA} \;\approx\; \sigma_1 \mathbf{u}_1\mathbf{v}_1^\top + \sigma_2 \mathbf{u}_2\mathbf{v}_2^\top + \cdots + \sigma_8 \mathbf{u}_8\mathbf{v}_8^\top$$

Gradient descent finds the $\mathbf{B}$ and $\mathbf{A}$ that minimize training loss, but the result is equivalent to keeping only the 8 dominant modes of the needed correction. Modes 9+ are below the noise floor.

Figure 6: The correction matrix decomposes into modes ordered by singular value (energy). LoRA keeps the 8 dominant modes. This is truncated SVD — structurally identical to keeping the fundamental modes of a vibrating system.

7.3 The Information Bottleneck

Matrix $\mathbf{A}$ is a measurement operator (compress $d$ dimensions to 8); matrix $\mathbf{B}$ is a reconstruction operator (project 8 dimensions back to $d$):

Figure 7: Inside each adapted layer. $\mathbf{A}$ compresses to an 8-dimensional “fingerprint” (what matters for the domain correction); $\mathbf{B}$ reconstructs a full-dimensional correction. The bottleneck forces the correction into a low-dimensional subspace.

8. Across Layers: The Feature Hierarchy

8.1 The CNN Precedent

The layer hierarchy was first understood through convolutional networks for vision, which recapitulate biological visual processing (Hubel & Wiesel, 1960s):

DepthCNN FeaturesBiological Vision
Layers 1–2Edges, oriented gradientsSimple cells in V1 — orientation-selective
Layers 3–5Textures, patterns, cornersComplex cells in V1/V2 — position-invariant
Layers 6–10Object parts, component shapesV4, inferotemporal cortex
Final layersObject identity, scene semanticsHigh-level recognition areas
The critical observation: early layers are universal (edge detectors transfer from dogs to X-rays); late layers are task-specific (the “golden retriever” classifier is useless for radiology). Transfer learning freezes early layers, retrains late layers. The same principle governs LoRA's choice of which transformer layers to adapt.
Figure 8: The universal → task-specific gradient is shared by CNNs and transformers alike. Domain specialization lives in the late layers; the early layers are foundational and don't need correction.

8.2 The Same Hierarchy in Transformers

DepthTransformer FunctionCNN Analogue
Layers 1–8Token embeddings, basic syntax, local patternsEdge detection — structural primitives
Layers 9–20Grammar, coreference, semantic relationshipsTexture/part recognition — compositional
Layers 21–28Abstract reasoning, domain knowledge, styleObject recognition — high-level meaning
Layers 29–32Task-specific output shapingClassification head — final decision

In hardware terms: you don't re-verify the entire board when you update application firmware. The lower levels (power, clocks, connectivity) are proven and stable. You test at the functional level — the top of the stack — where the change has effect.

9. Resolving the Apparent Paradox

In harmonic analysis, the fundamental modes (lowest frequency, most energy) come “first.” In neural networks, the most basic features are also in the first layers. But we adapt the LAST layers and call our rank-8 modes “fundamental harmonics.” Which end is fundamental?

The resolution: “fundamental” means different things on the two axes.

9.1 The Layer Axis: Compositional Hierarchy

Layers are ordered by abstraction level, not importance. Layer 1 isn't the most important — it's the most basic. Each layer composes the previous into higher abstractions:

$$\text{tokens} \;\xrightarrow{\text{layers 1–8}}\; \text{syntax} \;\xrightarrow{\text{layers 9–20}}\; \text{semantics} \;\xrightarrow{\text{layers 21–32}}\; \text{domain behavior}$$

You don't adapt the atoms to specialize the organism. The atoms are fine. “Fundamental” here means foundational — everything builds on it, but it doesn't need to change.

9.2 The Rank Axis: Energy Hierarchy

Within each adapted layer, modes are ordered by correction energy (singular value). Mode 1 carries the most — it's the dominant correction. This is the classical harmonic ordering:

$$\sigma_1 \;\geq\; \sigma_2 \;\geq\; \cdots \;\geq\; \sigma_8 \;\gg\; \sigma_9 \;\approx\; 0$$

“Fundamental” here means most energetic — the mode that carries the most correction.

9.3 The Axes Are Orthogonal

The layer axis is a compositional hierarchy (basic → abstract). The rank axis is an energy hierarchy (dominant → least). They are independent. The rank-8 modes are “fundamental” in the energy sense, operating within layers at the abstract end of the compositional hierarchy. No contradiction.

Part III — Connecting the Views

10. How the Outside Predicts the Inside

The two views are not independent observations — the thermodynamic (outside) description predicts the modal (inside) structure. Here is the logical chain:

1
The base model is at thermodynamic equilibrium outside
Pre-training annealed 3B parameters to a deep, stable energy minimum. The system is well-ordered — most degrees of freedom are tightly constrained to their equilibrium values.
2
Domain adaptation is a small perturbation to that equilibrium outside
“Hardware diagnostics specialist” is close to “general helpful assistant” in parameter space. Most of the model's structure doesn't need to change — only the domain-specific behavior.
3
Perturbation theory says: first-order corrections are low-rank bridge
When a perturbation is small relative to the system, the response is concentrated in a few modes. The correction's singular value spectrum decays rapidly. This is why $r = 8$ captures 27.5% val loss improvement — most of the correction energy lives in the first few modes.
4
The compositional hierarchy localizes the perturbation to the surface bridge
In any hierarchically structured system, perturbations to high-level behavior don't require changes to low-level structure. Syntax and grammar (deep layers) are shared across all domains; only the abstract, task-specific layers (surface) need correction.
5
Therefore: the correction is low-rank AND spatially localized inside
8 modes per layer (rank) × 8 adapted layers (surface) = 3.47M parameters. The macroscopic thermodynamic argument predicts both the rank structure and the layer localization that we observe empirically.
The punchline: The outside view (thermodynamics) and the inside view (normal modes) are not two separate analogies. They are the macroscopic and microscopic descriptions of the same physical system. The thermodynamic statement “the perturbation is small” is equivalent to the modal statement “the correction is low-rank and surface-localized.” One implies the other. This is exactly how statistical mechanics works: macroscopic thermodynamic properties emerge from microscopic modal structure.

11. The Complete Pipeline

1
Download base model
Import a pre-solved thermodynamic system ($\mathbf{W}_0$) annealed to equilibrium by Meta. Cost to create: millions. Cost to us: one download.
2
Quantize to 4-bit
Discretize the state space: $\mathbb{R} \to 2^4 = 16$ levels per weight. Minimal information loss — the system is at equilibrium. Memory: $\sim\!2$ GB.
3
Initialize LoRA adapters
Create the perturbation subspace: rank $r = 8$, top 8 layers. $\mathbf{B}$ initialized to zero (perturbation starts as “no change”).
4
Training (annealing)
Cool the perturbation parameters toward a domain-specific minimum. $\eta = 10^{-5}$ (cold), batch 1 (thermal noise), 600 iterations. The system finds the dominant correction modes via gradient descent.
5
Inference
$\mathbf{W} = \mathbf{W}_0 + \mathbf{BA}$ at each adapted layer. The ground state is perturbed by 8 learned modes per layer, producing domain-specialized output. The cathedral stands; the room is open.

12. Implications & Future Directions

12.1 What the Training Run Confirmed

Val loss $2.809 \to 2.037$ (27.5% reduction) with rank 8, top 8 of 32 layers, confirms:

  1. The correction is low-rank — 8 modes per layer suffice; the singular value spectrum decays rapidly.
  2. The correction is surface-localized — adapting 25% of layers (the abstract end) was enough.
  3. The two views are consistent — the small thermodynamic perturbation predicted both facts.

12.2 Tuning Strategies (the Two Axes)

StrategyWhat It DoesWhen to Try
Increase rank ($r = 16, 32$)More modes — finer correction structureDomain generally right but lacking nuance
More layers (16 of 32)Deeper perturbation into semantic layersNeed to change how the model reasons
More training dataMore signal for existing parametersDiminishing returns at current data size

Our loss curve showed 85% of improvement by iteration 200 — the current configuration has extracted most available signal from 385 examples. More data, not more parameters, is the next leverage point.
