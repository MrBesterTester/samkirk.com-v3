Uber Level AI Skills

Advanced techniques for getting dramatically more value from AI tools like ChatGPT, Claude, and Gemini. These are hard-won lessons from extensive real-world usage.

The Skill Hierarchy

AI proficiency can be broken into levels:

  1. Basic — Simple Q&A, accepting first responses
  2. Intermediate — Prompt engineering, iteration, context management
  3. Advanced — Multi-turn workflows, tool integration, output verification
  4. Uber — System design with AI, meta-prompting, autonomous workflows

Key Techniques

1. Context is King

The most important factor in AI output quality is the context you provide:

  • Give relevant background before asking questions
  • Include examples of desired output format
  • Specify constraints and edge cases upfront
  • Use structured formats (markdown, JSON) for complex inputs

Pro tip: Start complex sessions by having the AI summarize its understanding of the task before proceeding.

2. Iterative Refinement

Don't expect perfection on the first try:

  • Start with a rough draft, then refine
  • Ask the AI to critique its own output
  • Use follow-up prompts to address specific issues
  • Build up complex artifacts in stages

3. Verification Strategies

AI outputs need verification, especially for:

  • Factual claims (check sources)
  • Code (test before using)
  • Calculations (verify independently)
  • Edge cases (probe explicitly)

4. Workflow Design

For complex tasks, design your AI workflow:

  • Break large tasks into smaller, verifiable steps
  • Use different prompting strategies for different phases
  • Maintain artifacts (specs, outlines) between sessions
  • Know when to start fresh vs. continue iterating
